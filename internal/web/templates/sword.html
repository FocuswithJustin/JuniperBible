About SWORD Modules

SWORD (Software for the Organisation and Retrieval of Wayside Data) is an open-source Bible software framework. Modules are stored in a standard format with:

  • mods.d/ - Module configuration files (.conf)
  • modules/ - Module data files (zText, RawText, etc.)

Learn more at crosswire.org/sword
