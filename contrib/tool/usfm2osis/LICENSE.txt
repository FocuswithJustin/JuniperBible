usfm2osis is a Python tool for converting USFM (Unified Standard Format Markers)
files to OSIS (Open Scripture Information Standard) XML format.

Official Repository: https://github.com/chrislit/usfm2osis

This is a reference/documentation directory. For the actual software and its
complete license, please visit the official repository.

PyPI Package: https://pypi.org/project/usfm2osis/
